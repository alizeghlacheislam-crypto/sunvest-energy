# SUNVEST ENERGY — Landing Page

Static single-page website for SUNVEST ENERGY, a solar energy company based in Mila, Algeria. Arabic (RTL), dark premium design, zero build step.

## Structure

```
sunvest-energy/
├── index.html      # Full site (embedded CSS + JS)
├── favicon.png     # 32×32 favicon
└── assets/
    ├── logo.png
    ├── hero-installation.jpg
    ├── tech-inverter.jpg
    ├── tech-efficiency.jpg
    ├── card-residential.jpg
    └── card-industrial.jpg
```

## Local preview

Any static server works:

```bash
npx http-server -p 8123 -c-1 .
```

Then open <http://localhost:8123>.

## Deployment

Deployed on Vercel (framework preset: **Other** — no build step). Every push to `main` auto-redeploys.
